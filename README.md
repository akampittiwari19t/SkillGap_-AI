# 🤖 SkillGap-AI

SkillGap-AI is a simple Python application that helps users identify the skills required for their desired job role. The application compares the user's current skills with predefined required skills and displays the missing skills.

---

## 📖 Overview

SkillGap-AI is a beginner-friendly project built using **Python** and **Tkinter**. Users enter a target job role and their current skills. The application then compares the entered skills with the required skills for that role and shows the missing skills.

Currently, the project supports the following job roles:

* Data Analyst
* Cyber Security

This project is useful for students who want to understand which skills they need to learn for a particular career.

---

## ✨ Features

* Enter your target job role
* Enter your current skills
* Compare your skills with required skills
* Display missing skills instantly
* Simple and easy-to-use graphical interface
